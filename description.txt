This mod aims to improve the experience using Magic Storage when using
mods that add Post-Moon Lord content. Normally, the Moon Lord must be
farmed repeatedly to obtain enough Radiant Jewels. This mod adds the
Radiant Jewel drop to all bosses that are Post-Moon Lord, requiring less
Moon Lord farming.

This mod currently only supports Calamity, but I am looking to add
support for other popular mods.

Feel free to check out the source code:
https://github.com/rilstrats/RadiantJewelPostMoonLord
